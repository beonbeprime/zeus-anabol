O que o zeus-anabol instala
🧠
Personalidade com identidade propria
Voce da um nome, um estilo, uma area de foco. Ele para de ser generico e vira o seu parceiro.
🚫
Anti-lisonja ativo
Nao valida ideia fraca. Nao concorda por pressao. Fala o que voce nao quer ouvir quando precisa.
⚡
Autonomia total
Quando voce diz "faz": ele faz. Sem confirmacao em loop. Avisa depois, nao antes.
🔒
Gate de seguranca
Bloqueia automaticamente operacoes destrutivas e URLs financeiras sem autorizacao.
📦
FULLSAFE versionamento
Antes de editar qualquer arquivo, cria copia de seguranca. Se der errado, rollback imediato.
📚
Aprendizado continuo
Cada erro vira uma regra. Sessao a sessao fica mais inteligente sobre voce e seu negocio.
🗂️
Memoria persistente
Lembra do que importa entre as sessoes. Tom de voz, negocios, preferencias, decisoes.
✅
Gate de entrega
Nunca entrega URL sem verificar. Nunca afirma que funciona sem ter testado.

Como instalar (5 minutos)
  • 1
    Clone o repositorio No terminal, rode: git clone https://github.com/allyssonsilveira/zeus-anabol
  • 2
    Entre na pasta cd zeus-anabol
  • 3
    Rode o onboarding interativo node onboarding/setup.js — responde 7 perguntas e gera sua personalidade
  • 4
    Abra o Claude Code Ainda na mesma pasta: claude
  • 5
    Fale com ele como parceiro Nao como ferramenta. Nao como suporte. Como socio que conhece o seu negocio.
