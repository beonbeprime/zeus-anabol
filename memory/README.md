# Sistema de Memoria

Esta pasta armazena a memoria persistente do seu Claude Code.

## Arquivos que serao criados aqui

- `perfil.json` — dados do seu onboarding (gerado pelo setup.js)
- `erros-aprendidos.md` — log de erros e as regras criadas para evitar repeticao
- Outros arquivos de memoria conforme voce usar o Claude Code

## Como a memoria funciona

O Claude Code le automaticamente os arquivos desta pasta no inicio de cada sessao.
Quanto mais voce usar, mais ele aprende sobre voce e sobre o seu negocio.

## Dica

Voce pode pedir ao Claude:
- "lembra que eu falei X ontem?" — ele busca na memoria
- "salva isso na memoria" — ele cria um arquivo aqui
- "o que voce ja aprendeu sobre mim?" — ele lista os arquivos
